Windows Setup

  1. Press Win + R, type C:\Windows\System32\drivers\etc
  2. Edit the file hosts with Notepad
  3. Add these lines:
51.79.180.62 growtopia1.com
51.79.180.62 growtopia2.com
51.79.180.62 www.growtopia1.com
51.79.180.62 www.growtopia2.com
  1. Save the file, then launch Growtopia
  2. Enjoy playing Skytopia

iOS Setup

  1. Download Surge 5 from the App Store
  2. Open Default.conf, tap Import Profile, and paste the host URL
    3. https://dash.gtps.cloud/ios/1841
    1. Allow the VPN setup, then open Growtopia
    2. Enjoy playing Skytopia

Mac Setup

  1. Edit the /etc/hosts file just as on Windows
  2. Add the same server IP addresses
  3. Save, launch Growtopia, and connect to Skytopia
